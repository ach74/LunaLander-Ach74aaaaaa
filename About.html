Información sobre el proyecto

Introducción

Este juego ha sido creado por Achraf El Ouakili, del curso de desarrollo de aplicaciones multiplataformas. Para la asignatura de lenguaje de marcas.

Creador del juego: Achraf El Ouakili.
Destinatario: Miguel Angel Urbina Pons.
Objetivos del juego: La finalidad de este juego es aterrizar en la base lunar a una velocidad moderada.

Desarrollo

Este proyecto ha sido creado con código HTML,CSS y JavaScript. Y para poder solventar los inconvenientes de distintas pantallas, el juego es adaptable mediante dos CSS. Donde uno esta hecho para usar en pantallas de Ordenadores, tables con una pantalla considerable ... y otro CSS para dispositivos más pequeños como móviles. (esto está explicado con más detalle en el README)
